в зависимости от данного числа. Использовать javascript.

*v2,1,2,3,...* мешков с *s2*.Сколко мешков осталось в *s1*?

Разработать формат строки для алгоритма решения рандомной задачи.
Пример:
D1=v1+v2,D2=v1-v3.

тестов.
